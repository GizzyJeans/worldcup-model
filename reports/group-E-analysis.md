# 世界盃 2026 — E 組末輪完整分析報告

分析方法:模型(Dixon-Coles + Elo)算出預期進球與機率,再與多家莊家的「銳利
共識」(Pinnacle/Betfair/Smarkets 去除水位後)混合(w=0.9,即 90% 市場 + 10%
模型),最後對全市場最佳賠率比價。晉級率來自 20,000 次蒙地卡羅模擬(含最佳
第三名規則)。

### 詳細判讀 → 建議「跳過」
1. **庫拉索防守是全屆最差之一**:首輪被德國 7-1、再 0-1,兩場丟 8 球。
2. **象牙海岸有動機**:贏球可鎖第 2(晉級率 97% 但未到 100%)。

4. 因此「庫拉索 +2.25」「小 3.25」「和局」這些 +EV 注,全是「**象牙海岸不會
   大勝**」的賭注 —— 而上述趨勢強烈反對。
5. **結論**:帳面 +EV 在「反血洗」側,是本屆連敗的角度;真要順勢只能買象牙海岸
   大勝,但賠率被壓到 −EV。→ **無乾淨價值,整場 PASS。**

### 動機分析(模型看不到的關鍵)
- **德國 = 內容方**:1st 已鎖、GD+7,贏了沒有更好 → **零動機,可能大輪換**
  保留主力給淘汰賽。
- **厄瓜多 = 急迫方**:必須贏才有較穩的晉級機會(目前 26.5%)→ 全力以赴。

### 為何這場與庫拉索場相反、可以下
關鍵差異是**天賦差距**。我整理本屆「急迫弱方 vs 內容強方」的案例:

| 對戰 | Elo 差距 | 結果 |
|---|---|---|
| 捷克(急) vs 墨西哥(內容) | 281 | 墨西哥 3-0 血洗(差距太大) |
| 南非(急) vs 南韓(內容) | 227 | **南非 1-0 爆冷**(差距中等) |
| **厄瓜多(急) vs 德國(內容)** | **76** | **差距最小 → 最有利急迫方** |

「站急迫方」的下注前提(差距不能太大)在這場**完全成立**:
1. ✅ 德國真安全 → 零動機、可能輪換
2. ✅ 厄瓜多非贏不可 → 拚命
3. ✅ **天賦接近**(差 76,厄瓜多是世界排名相近的真強隊)
4. ✅ 低分對戰(2.33)→ 利於弱方咬住

### 風險

- 「fade favorite」本屆已多次失敗 → **中等注、別重壓**。
- 但若「急迫 + 小差距」這條件有效,這是全屆最該下的一次。

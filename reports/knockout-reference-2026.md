## 🎯 進入淘汰賽的下注原則(小組賽實戰驗證後)

### 淘汰賽 vs 小組賽:最大差異
- **沒有平局**(90 分鐘平 → 延長賽/PK)→ 1X2 變成「誰晉級」;讓分/大小仍以 90 分鐘結算。
- **每隊都是輸即淘汰 → 全員全力**。小組賽那些**動機模板全部退役**:
  「默契和局」「死局輪換」「best-third 急迫」「content favorite」在淘汰賽都不存在。
- → **淘汰賽回歸純實力(Elo/xG)+ 線上比價(line-shopping)+ 進球邏輯。**

### 核心原則(依重要性)
1. **模型無 alpha,一律 blend to market。** 模型打不贏 sharp 收盤線;唯一 edge 是**比價軟盤**,用 CLV 衡量。
2. **實力差是淘汰賽主訊號。** 沒有動機局可吃 → Elo/xG 差距 + 防守數據說話。
3. **🚨 別過度押小分(本屆最大、最常重犯的錯)。** 末三輪幾乎場場 OVER,小分注被血洗。
   - **凡有一方需要進球 → 絕不押小分**(塞內加爾 5-0、剛果 3-1、英格蘭 2-0 全打臉小分)。
   - **淘汰賽落後方必須追分 → 後段易開盤** → 大分/讓盤優先於小分。
4. **強隊就算「不絕望」照樣靠實力贏。**(克羅埃西亞 2-1)→ 淘汰賽別用動機想太多,先看實力。
5. **讓分紀律:**
   - 強隊淺讓盤(−0.25/−0.75):跟市場,別反買。
   - 深讓盤(−1.5/−2):只在「強隊 + 需要/會進球 + 對手弱」時追;對手鐵桶陣(摩洛哥、迦納曾零失球)或大熱門無動機(阿根廷對約旦 1-3 仍開放)時別追。
   - 深受讓(+2.25/+2.5):別預設是血洗陷阱,逐場看 cover 機率。
6. **PK 變數:** 90 分鐘平手機率高的場,「晉級」遠比 90 分鐘盤口更不確定 → 大熱門的「晉級」賠率常被高估。

### 模型校準現況(供下注權重)
- 小組賽「急迫弱方 + Elo 差 < ~150 + 強隊真鬆懈」模板 3-1(淘汰賽不再適用)。
- 結構/動機方向讀得準(高賠平局、best-third 預測幾乎全中);**弱點是系統性偏小分** → 淘汰賽刻意上修進球預期。
